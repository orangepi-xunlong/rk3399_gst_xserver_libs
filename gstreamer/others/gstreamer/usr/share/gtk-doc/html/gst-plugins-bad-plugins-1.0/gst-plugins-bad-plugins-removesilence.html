Description

Removes all silence periods from an audio stream, dropping silence buffers.

Example launch line

1
 
gst-launch-1.0 -v -m filesrc location="audiofile" ! decodebin ! removesilence remove=true ! wavenc ! filesink location=without_audio.wav
